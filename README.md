# node-redis
## NodeJS & REDIS
to setup this application:
* clone the repo
* change directory
* run `npm install`
* run `npm start`
* visit localhost:3000

## Body Parameters
`first_name`
`last_name`
`phone`
`email`
`id`

## Using Postman
* add user: /user/add
* delete user: /user/delete/{id}
* get a user: /user/{id}
* update a user: /user/update/{id}
